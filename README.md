# kojo-nav-group-prototype
Prototype for kojo's top bar using html popover and position anchor

## SCRIPTS
### Run for development
1. `nvm use`
2. `npm run dev`

### Build
1. `nvm use`
2. `npm run build`

### Deploy to github pages
1. `nvm use`
2. `npm run deploy`
